Map Sprinkles
/datum/nmnode/mapload/sprinkles

Similar to variations mode, but rolls all files individually rather than picking one, using name for landmark. The prefix number is used as a percentage chance.

Example: some/folder/10.something_funny.dmm would have 10% chance to insert at the 'something_funny' landmark
